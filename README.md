# Team Task Manager 📋

Full-stack web app for project & task management with role-based access control.

## Features
- 🔐 JWT Authentication (Signup/Login)
- 👥 Role-based access (Admin / Member)
- 📁 Project & team management
- ✅ Task CRUD with assignment, priority, due dates
- 📊 Dashboard with task stats (Todo/In Progress/Done/Overdue)
- 🎯 REST APIs with proper validations

## Tech Stack
- **Frontend:** React, React Router, Axios
- **Backend:** Node.js, Express, JWT
- **Database:** MongoDB (Mongoose)
- **Deployment:** Railway
